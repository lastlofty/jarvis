# Плагин websearch — поиск в интернете

Скилл `web_search(query)` ищет в интернете через DuckDuckGo (бесплатно, без ключа).

Требует пакет: `pip install ddgs`. Если пакет не установлен — скилл вернёт
подсказку, не ломая агента.

## Примеры команд
- «Найди в интернете последние новости про GLM»
- «Поищи, сколько стоит RTX 4070»
