# Плагин wikipedia

Добавляет агенту Jarvis скилл `wikipedia_summary` — поиск кратких справок в
Википедии через бесплатный REST API (ключ не нужен).

## Что умеет
- Принимает тему/вопрос и язык (`ru` или `en`).
- Находит наиболее релевантную статью и возвращает её краткую выжимку и ссылку.

## Как устроен плагин
- `manifest.json` — метаданные плагина и список файлов знаний.
- `skill.py` — объявляет `DECLARATIONS` (схема инструмента для модели) и
  `HANDLERS` (реализация). Загрузчик плагинов регистрирует их автоматически.
- `README.md` — этот файл; он индексируется в базу знаний (RAG), поэтому агент
  «знает» о существовании и назначении плагина.

Это эталон, по которому можно делать свои плагины: создайте папку в `plugins/`,
положите `manifest.json`, при необходимости `skill.py` со скиллами и любые
`.md`/`.txt` с знаниями — Jarvis подхватит всё при старте.
