# Плагин call_analyzer — анализ звонков и CRM

AI-агент для анализа звонков: читает транскрипт разговора, готовит саммари и
автоматически заполняет карточку в локальной CRM.

## Скиллы
- `read_transcript(path)` — прочитать транскрипт (txt/pdf)
- `save_crm_record(client, summary, action_items, status, next_step)` — сохранить карточку
- `list_crm_records()` — показать карточки

## Как работает
Агент читает транскрипт, сам делает выжимку (саммари, договорённости, статус,
следующий шаг) и вызывает `save_crm_record`. CRM хранится в
`<SAFE_ROOT>/crm/records.json`.

## Пример команды
«Проанализируй звонок call.txt, сделай саммари и заведи карточку в CRM»
