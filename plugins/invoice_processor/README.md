# Плагин invoice_processor — счета и финансовая отчётность

AI-система для обработки счетов: учёт, сверка данных и подготовка отчётности.

## Скиллы
- `read_invoice(path)` — прочитать счёт (txt/pdf)
- `record_invoice(number, vendor, date, amount, currency)` — записать в реестр
- `reconcile_invoices()` — сверка: дубликаты, некорректные суммы
- `financial_summary()` — итоги по поставщикам и валютам

## Как работает
Агент читает счёт, извлекает поля (номер, поставщик, дата, сумма) и вызывает
`record_invoice`. Реестр — `<SAFE_ROOT>/finance/invoices.csv`. Затем можно сверить
данные и получить сводку. Запись дубля (тот же номер+поставщик) блокируется.

## Пример команды
«Обработай счёт invoice_123.pdf, занеси в реестр, потом сделай сверку и сводку»
