# Плагин vision — зрение Jarvis

Даёт агенту способность «видеть» через бесплатную мультимодальную модель
`glm-4v-flash` (Zhipu / BigModel). Ключ берётся из `GLM_API_KEY`.

## Скиллы
- `analyze_screen(question)` — делает скриншот экрана и отвечает на вопрос о нём.
- `analyze_image(path, question)` — анализирует изображение по пути к файлу.

## Примеры команд
- «Что сейчас на экране?»
- «Посмотри на экран и скажи, есть ли ошибки в коде»
- «Что изображено на C:/Users/.../photo.jpg?»
