# Плагин image_gen — генерация изображений

Скилл `generate_image(prompt)` создаёт картинку по текстовому описанию через
бесплатную модель `cogview-3-flash` (Zhipu / BigModel) и сохраняет её в
`<SAFE_ROOT>/jarvis_images/`.

## Примеры команд
- «Нарисуй кота-киберпанка в неоновом городе»
- «Сгенерируй логотип для проекта Jarvis в стиле терминала»
