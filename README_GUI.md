# 🎨 Jarvis GUI (PySide6)

Полноценное desktop-приложение в стиле Steam/Discord — тёмно-синяя тема, кастомный заголовок, системный трей.

## 📋 Установка GUI поверх существующего проекта

Этот архив добавляется поверх уже работающего Jarvis (того, у которого ты уже починил ключ Gemini).

### 1. Распакуй архив в корень проекта

```
C:\Users\ohotn\OneDrive\Рабочий стол\jarvis\
├── src\jarvis\          ← уже есть
│   ├── core\
│   ├── llm\
│   ├── ...
│   └── gui\             ← НОВОЕ из архива
│       ├── __main__.py
│       ├── main_window.py
│       ├── theme.py
│       ├── bridge.py
│       ├── pages\
│       └── widgets\
├── jarvis_gui.spec      ← НОВОЕ
├── build_gui.bat        ← НОВОЕ
└── requirements_gui.txt ← НОВОЕ
```

### 2. Установи PySide6

В активном venv:

```powershell
pip install -r requirements_gui.txt
```

(Установка ~100 МБ, займёт минуту-две)

### 3. Запусти GUI напрямую (без сборки exe)

```powershell
python -m jarvis.gui
```

Должно открыться окно: чёрно-синяя тема, слева — кнопки «💬 Чат / 🖥 Экран ПК / ⚙ Настройки», сверху — кастомный заголовок с твоими кнопками минимизации и закрытия.

Перетаскивание окна работает за заголовок. Двойной клик по заголовку — максимизация. Закрытие — сворачивает в трей (правый-нижний угол Windows).

### 4. Собери exe

```powershell
.\build_gui.bat
```

Сборка идёт 2-4 минуты. На выходе:

```
dist\Jarvis\
├── Jarvis.exe          ← запускается двойным кликом
├── _internal\          ← ~150 МБ DLL и пакетов
└── .env.example
```

**Перед первым запуском exe:**

```powershell
Copy-Item .env dist\Jarvis\.env
```

Без `.env` не будет работать ключ Gemini.

Затем:

```powershell
dist\Jarvis\Jarvis.exe
```

Или просто двойной клик из проводника.

## ✨ Что есть в GUI

| Возможность | Где |
|---|---|
| Чат с Jarvis (Enter — отправить, Shift+Enter — перенос) | Вкладка «Чат» |
| Индикатор «Jarvis думает...» с анимацией | внизу чата |
| Скриншот рабочего стола с авто-обновлением каждые 2 сек | «Экран ПК» |
| Редактор `.env` прямо из приложения | «Настройки» |
| Сворачивание в системный трей | крестик в углу |
| Перетаскивание окна за заголовок | кастомный title-bar |
| Адаптация к ресайзу | везде |

## 🐛 Если упадёт

| Симптом | Решение |
|---|---|
| `ModuleNotFoundError: No module named 'PySide6'` | `pip install -r requirements_gui.txt` |
| `ModuleNotFoundError: No module named 'jarvis.gui'` | Распакуй архив в правильное место (см. дерево выше) |
| Окно открылось, но при первом сообщении — ошибка про API key | Тот же ключ, что в консольной версии. Поправь его в Настройках или в .env |
| После сборки `ImportError` в exe | Добавь имя пакета в `HIDDEN_IMPORTS` в `jarvis_gui.spec` и пересобери |
| Окно белое / без стилей | Не загрузился QSS. Запусти из консоли — увидишь traceback |

## 🎨 Поменять цвета темы

Открой `src/jarvis/gui/theme.py` — все цвета в классе `Colors`. Поменяй `ACCENT` с `#1e88e5` на что хочешь и пересобери.
